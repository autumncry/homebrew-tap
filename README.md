# Homebrew Tap for NetStats

This tap provides the Homebrew Cask for [NetStats](https://github.com/autumncry/netstats), a native macOS menu bar monitor for CPU, memory, network speed, public IP, geolocation, and Clash Verge Dev status.

## Install

```bash
brew tap autumncry/tap
brew install --cask netstats
```

## Upgrade

```bash
brew update
brew upgrade --cask netstats
```

## Uninstall

```bash
brew uninstall --cask netstats
brew untap autumncry/tap
```

## Notes

The current public NetStats DMG is unsigned. macOS may block the first launch. Open **System Settings > Privacy & Security** and allow NetStats, or Control-click the app and choose **Open**.
